# 应该使用哪个版本？


建议始终运行最新版本的 YuniKorn。新版本定期由 YuniKorn 本身和/或 shim 的新功能驱动。目前新功能和修复仅合并到用来做释出的开发分支。

此时该项目仅创建次要版本，没有补丁版本。

如果 shim 版本（即 Kubernetes）受多个 YuniKorn 版本支持，我们建议始终使用最新的 YuniKorn 版本。
早期版本中较旧的构建依赖项可能会阻止您使用 shim 的某些新功能。
